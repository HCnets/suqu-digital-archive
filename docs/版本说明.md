# 苏区镇数字化档案 - 版本更新与技术说明

> ⚠️ **规范警告**: 本文档中的历史版本说明禁止删除，仅允许向下追加，以形成完整的技术演进时间树。

## [v0.1.0-alpha] - 2026-05-16
### 1. 架构选型 (GPT-5.4 推荐)
- **WebGIS 引擎**: 推荐使用 Cesium.js 处理宏观地形（DEM）与倾斜摄影模型，或 Three.js 结合 MapboxGL 处理轻量级建筑级交互。
- **前端框架**: React 18 + TypeScript + Vite，保障极速编译与模块化。
- **状态管理**: Zustand（轻量且适合处理复杂的 3D 状态交互）。

### 2. UI 规范 (Gemini 3.1 Pro Max 设定)
- **大一统 Header**: 采用横向流式布局，左侧为大字号标题（headlineMedium），右侧为独立“返回”按钮。
- **返回按钮设计**: 尺寸 20dp icon，12dp padding，带有 Liquid Glass (液态玻璃) 质感。
- **信息面板**: 点击地图上的 3D 建筑/热点时，弹出的档案信息面板必须采用磨砂玻璃效果（backdrop-filter: blur），突出“数字化”与“未来感”。
