# 广东省苏区镇数字化档案 (Suqu Town Digital Archive)

## 📌 项目概述
本项目旨在打造一个高精度的广东省苏区镇数字化档案库。通过 **地图点击建模 (Map-Clicking 3D Modeling)** 技术，实现宏观地理信息与微观建筑/历史档案的无缝交互。
