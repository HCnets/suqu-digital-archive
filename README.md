# 广东省苏区镇数字化档案 (Suqu Town Digital Archive)

## 📌 项目概述
本项目旨在打造一个高精度的广东省苏区镇数字化档案库。通过 **地图点击建模 (Map-Clicking 3D Modeling)** 技术，实现宏观地理信息与微观建筑/历史档案的无缝交互。

## 🌟 项目纪年表与版本演进 (Time Tree)
*注：以下纪年表严格遵循时间树规范，只增不减，完整体现版本工程推进及成果。*

### [v0.4.0-beta] - 2026-05-16
- **版本状态**: 时间沙盘与多源底图架构
- **工程推进**:
  - 新增 `TimeSlider` 全局底部时间轴组件，支持 1920-2030 年份拖拽，档案点随时间线动态显示/隐藏。
  - 新增 Map Style Switcher (图层切换器)，支持在 `Carto Dark` (科技暗黑) 和 `ArcGIS World Imagery` (高清卫星实景) 之间无缝切换。
  - 引入模拟的 `GeoJSON` 边界数据，在地图上绘制苏区镇半透明发光行政边界 (Polygon)。
- **阶段成果**:
  - 用户现在可以通过拖拽底部时间轴，直观地看到苏区镇历史档案的演进过程。
  - 随时切换至卫星图查看历史事件发生的真实地貌环境。
  - 确立了苏区镇的数字版图边界。

### [v0.3.0-beta] - 2026-05-16
- **版本状态**: 大屏级交互与多媒体档案下钻
- **工程推进**:
  - 开启 `MapLibre` 的 `3D Terrain` (DEM 地形) 与 `fill-extrusion` (3D 建筑拉伸) 支持。
  - 数据源 (`store/index.ts`) 扩展 `year` (年份)、`content` (富文本)、`media` (多媒体相册) 字段。
  - 新增 `ArchiveDetailModal` 全屏沉浸式档案模态框。
  - 实现基于 `setInterval` 和 `map.flyTo` 的“自动巡航”漫游功能。
- **阶段成果**:
  - 拥有了真实起伏的山体地形。
  - Header 增加“自动巡航”按钮，点击后镜头自动在各大历史档案点之间平滑穿梭。
  - 点击“深度解析档案”，弹出带有大图相册、年份标识的磨砂玻璃详情页。

### [v0.2.0-beta] - 2026-05-16
- **版本状态**: 真实地理坐标系 GIS 架构升级
- **工程推进**:
  - 弃用纯虚构 3D 坐标空间，引入 `maplibre-gl` 和 `react-map-gl` 作为真实地理底座。
  - 确立以“广东省紫金县苏区镇 (LNG: ~115.34, LAT: ~23.36)”为核心的数字孪生坐标系。
  - 基于 CSS 3D Transform 实现无模型依赖的数据驱动发光光柱 (PoiMarker)。
- **阶段成果**:
  - 实现真实科技暗黑风格底图加载。
  - 数据层 (Store) 扩展，支持经纬度字段与档案类型分类 (政府/革命/文化)。
  - UI 层动态渲染分类颜色标识，档案面板增加经纬度数据回显。
